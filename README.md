Next.js + GraphQL + Typescript frontend boilerplate.

# Get started
- `git clone ...`
- `npm install`
-  `cp .env_example .env`
- Edit `.env`
- `npm run gen`
- `npm run dev`

## Types generation
Run `npm run gen` after every GraphQL API Schema changed or after `./graphql/*.graphql` files are modified
